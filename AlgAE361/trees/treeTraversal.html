CS361 Tree Traversals

Quick instructions:

  • Select a traversal from the Algorithm menu.
  • Use the "step" button to single-step through the code or use the "play" button to run all the way to the end of the calculation.
  • If you find the tree display is too hopelessly tangled, you can use your mouse to drag the nodes to more appropriate positions whenever you are paused within a traversal algorithm.

    • More detailed help is available from the Help menu.
